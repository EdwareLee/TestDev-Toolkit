## 正则匹配

💡 总结：常用正则符号速查
表格
符号	含义
\d	数字（0–9）
\w	字母、数字、下划线
.	任意字符（除换行）
*	0 次或多次
+	1 次或多次
?	0 次或 1 次
{n}	恰好 n 次
{m,n}	m 到 n 次
()	捕获组（用 .group(1) 取）
\b	单词边界
\.	匹配真正的点号
---
\d 表示一个数字（0–9）
{4} 表示前面的 \d 重复 4 次 → 年份（如 2026）
- 和 : 是普通字符，直接匹配
所以整体匹配：四位年-两位月-两位日 空格 两位时:两位分:两位秒

\d{1,3}：1 到 3 位数字（如 1、192）
\.：匹配点号（注意要加反斜杠，因为 . 在正则中是“任意字符”，所以要转义）
\b：单词边界（防止匹配到 192.168.1.1055 这种超长数字）

code:：字面匹配
\s*：匹配 0 个或多个空白字符（空格、tab 等）
(E\d+)：括号表示捕获组，我们只关心这部分
E：字母 E
\d+：1 个或多个数字
